# salary-management-mvp

## Setup

1. Open terminal, 
    ```
    cd node-app 
    
    #if pm2 is installed
    pm2 start app.js
    
    #else
    node app.js
    ``` 
2. Open new terminal, 
    ```
    cd salary-react-app 
    npm install # only for first time setup
    npm start
    ```
App will run on localhost:3000
